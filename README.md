# Forecasting Aplicado a Logística

Este curso introduce al equipo de logística en el uso práctico del forecasting para la planeación de inventarios en bodegas. A través de ejercicios aplicados, los participantes aprenderán a analizar datos históricos, generar pronósticos en Excel, medir su precisión y visualizar tendencias. Finalmente, explorarán el uso de herramientas como Power BI Desktop (gratuito) e IA generativa para apoyar decisiones operativas y escenarios de demanda.

## Estructura

- `CapituloXX/README.md`: guía de laboratorio por capítulo.
